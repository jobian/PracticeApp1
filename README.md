# PracticeApp1

Test Android mobile app for programmatically collecting cell site measurements.

## Passive measurement data
- Timestamp (milliseconds)
- RSRP
- RSRQ
- RSSNR
- CQI

## Active measurement data
- DL Throughput
- Latency
- Packet loss

## Cell site identification
- MCC
- MNC
- TAC
- PCI
- eNB ID
- Band
